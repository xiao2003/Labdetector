# GitHub 发布与普通用户下载说明

版本：1.0.0

## 当前正式发布物

当前仓库正式版仅保留以下两个发布文件：

1. `NeuroLab-Hub-v1.0.0.zip`
2. `NeuroLab-Hub-source-v1.0.0.zip`

说明：

- `NeuroLab-Hub-v1.0.0.zip` 面向普通用户，提供轻量入口与运行时下载机制。
- `NeuroLab-Hub-source-v1.0.0.zip` 面向开发者，用于二次开发、联调与自定义部署。
- 本地语音模型、本地大模型和其他大体积运行资产不再随发布包直接分发，而是在首次启动后的自检阶段或首次使用相应能力时按需下载。

## 发布原则

1. GitHub Release 仅保留当前正式版，不保留旧测试版包体。
2. 仓库源码与 Release 资产的版本号必须统一为 `1.0.0`。
3. 普通用户下载后即可进入启动引导，自检会自动补齐缺失的运行依赖、语音模型和本地模型框架。

## 普通用户使用方式

1. 下载 `NeuroLab-Hub-v1.0.0.zip`
2. 解压后运行 `pc/NeuroLab Hub.exe`
3. 按启动引导进入系统自检
4. 等待程序自动补齐缺失的依赖、语音模型与本地模型
5. 完成后进入主界面使用

## 开发者使用方式

1. 下载 `NeuroLab-Hub-source-v1.0.0.zip`
2. 解压后按 `README.md` 与 `docs/NeuroLab_Hub_完整手册.md` 配置环境
3. 使用源码方式启动、联调、测试与打包
