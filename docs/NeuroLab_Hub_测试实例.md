# NeuroLab Hub——可编排专家模型的实验室多模态智能中枢 测试实例

版本：1.0.0  
编写日期：2026 年 3 月 22 日

## 一、说明

本文件整理 NeuroLab Hub 正式版 1.0.0 的核心测试实例，覆盖 GUI、知识库、语音、训练、单节点闭环与多节点闭环等场景。

## 二、测试实例列表

### T01 主程序 GUI 启动

- 目标：验证主程序可正常进入启动页和工作台界面
- 预期结果：主界面正常显示，布局和主功能按钮可用

### T02 系统自检

- 目标：验证自检可发现并补齐运行时依赖
- 预期结果：缺失依赖、语音模型和本地模型框架可按需补齐

### T03 知识库导入

- 目标：验证 GUI 可将外部知识导入 `common` 或 `expert.*`
- 预期结果：导入成功并可在后续推理中使用

### T04 本地模型推理

- 目标：验证 `Ollama` 本地模型在 GUI 中可正常工作
- 预期结果：可选择模型并完成问答或解释

### T05 PC 本地语音交互

- 目标：验证 PC 端语音唤醒、控制、问答和记录
- 预期结果：支持唤醒、停止播报、结束本轮和知识提取

### T06 单节点闭环

- 目标：验证单个 Pi 节点的视觉和语音闭环
- 预期结果：Pi 上报 -> PC 分析 -> 回传 Pi 播报链路成立

### T07 多节点闭环

- 目标：验证多节点并发接入和日志隔离
- 预期结果：多个节点可同时在线，日志分离清晰，回传独立

### T08 聚焦闭环专家

- 目标：验证着装安全、危化品识别和语音记录三条聚焦闭环
- 预期结果：三条链路可用，其他专家不干扰主闭环

### T09 训练工作台

- 目标：验证视觉训练和 LLM 工作台入口可打开
- 预期结果：工作台界面正常，训练链路不报错
