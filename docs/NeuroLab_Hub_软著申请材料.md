## 三、可直接复用的软件简介

NeuroLab Hub 是一套面向实验室场景的中心-分布式多模态智能系统。系统由 PC 中心端、Pi 边缘节点、专家模型体系、多知识库体系和训练工作台构成，支持视觉识别、知识增强、语音交互、实验档案管理和模型训练回灌，可用于实验室安全监测、危化品识别、知识沉淀和智能辅助分析。

## 四、可直接复用的主要功能说明

1. 多节点实验视频接入与集中监控
2. 危化品识别、着装安全和专家分析
3. PC 与 Pi 双端语音交互
4. 公共知识库与专家知识库管理
5. 实验档案记录与结果回传
6. LLM 工作台与视觉训练工作台

## 五、技术特点说明

1. 采用 PC 中心端与 Raspberry Pi 边缘节点协同架构
2. 采用专家模型 + 知识库 + 统一多模态解释的组合链路
3. 支持本地模型和云端模型接入
4. 支持轻量交付包与运行时按需下载依赖和模型
5. 支持面向真实实验数据的训练闭环
