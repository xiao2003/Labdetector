# LabDetector 用户手册

## 1. 软件定位

LabDetector 是一套面向实验室场景的智能监控与语音交互软件，包含：

- PC 端桌面控制中心
- Pi 端边缘采集与交互节点
- 专家模型管理
- 多知识库管理
- 实验档案中心
- 训练工作台

## 2. 普通用户如何使用

普通用户不需要下载源码，直接进入 GitHub Release 页面下载安装器：

- Release 页面：<https://github.com/xiao2003/Labdetector/releases>
- 下载文件：`LabDetector-Setup-vX.Y.Z.exe`

双击安装器后即可：

- 阅读许可协议
- 自定义安装目录
- 创建桌面快捷方式
- 完成安装后启动软件

## 3. 安装后的目录

安装后软件目录通常为：

```text
安装目录
├─ LabDetector.exe
└─ APP
```

其中 `APP` 为运行时目录，包含程序依赖、文档、资源与配置。

## 4. 主要功能

### 4.1 监控会话

支持两种模式：

- 本机摄像头模式
- 树莓派集群 WebSocket 模式

### 4.2 专家模型与知识库

- 显示中文专家名称
- 支持导入专家模型资产
- 支持导入公共知识库与专家专属知识库
- 支持文本、图片、音频、视频资料导入

### 4.3 实验档案中心

- 按项目、实验、人员、时间归档运行记录
- 保存关键事件、专家结论与关键帧

### 4.4 训练工作台

训练工作台支持：

1. 导入真实 LLM 微调数据
2. 导入真实 Pi 检测数据
3. 自动构建训练工作区
4. 启动 LLM LoRA 微调
5. 启动 Pi 检测模型微调
6. 一键运行完整训练流程

## 5. 开发者说明

开发者使用源码仓库：

```powershell
git clone https://github.com/xiao2003/Labdetector.git D:\Labdetector
cd D:\Labdetector
```

常见打包命令见 [EXE与安装包生成教程.md](EXE与安装包生成教程.md)。
