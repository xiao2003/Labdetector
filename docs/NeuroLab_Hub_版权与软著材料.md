## 二、软著登记推荐口径

### 2.1 建议登记名称

- 软件全称：NeuroLab Hub——可编排专家模型的实验室多模态智能中枢软件
- 软件简称：NeuroLab Hub
- 版本号：V1.0.0

### 2.2 软件简介口径

NeuroLab Hub 是一套面向科研实验室场景的多模态智能中枢软件系统。系统采用 PC 中心端与 Raspberry Pi 边缘节点协同架构，融合计算机视觉、语音交互、知识库增强、专家模型编排、实验档案沉淀和模型训练能力，支持实验室监控、风险预警、智能问答、知识沉淀与训练闭环。

### 2.3 功能口径

建议对外统一描述为：

1. 多节点实验视频采集与集中可视化监控
2. 关键帧识别与专家分析
3. 风险提示、告警语义化与结果播报
4. 公共知识库与专家知识库管理
5. Pi 端与 PC 端语音智能交互
6. 实验过程档案记录与对话归档
7. LLM 微调与识别模型训练工作台
8. 模型结果回灌与持续迭代

### 2.4 技术特点口径

建议统一为：

- PC 中心端 + Raspberry Pi 边缘节点协同架构
- 专家模型体系与知识库体系结合
- 固定管家层 + 用户可选执行层的双层模型架构
- 支持本地与云端执行层模型接入
- 支持实验过程结构化归档与训练回灌
- 以 Windows 安装包与 Pi 目录复制部署方式交付
